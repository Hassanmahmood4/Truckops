# UI Guidelines

TruckOps UI follows a monochrome SaaS style:

- Colors: black, white, gray shades only
- Layout: roomy spacing with `max-w-6xl` containers
- Components: rounded corners, subtle borders, minimal shadows
- Motion: soft and restrained (no flashy transitions)

## Core Principles
1. Keep interfaces calm and predictable.
2. Prioritize readability and whitespace.
3. Use consistent typography hierarchy.
