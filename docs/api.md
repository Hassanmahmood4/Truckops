# API Endpoints

All endpoints are served from the Express API under `/api`.

## Drivers
- `POST /api/drivers`
- `GET /api/drivers`
- `PUT /api/drivers/:id`
- `DELETE /api/drivers/:id`

## Loads
- `POST /api/loads`
- `GET /api/loads`
- `PUT /api/loads/:id`

## Assignments
- `POST /api/assignments`
- `GET /api/assignments`

## Quotes
- `POST /api/quotes`
- `GET /api/quotes`
