# Local Setup

## Prerequisites
- Node.js 20+
- npm 10+

## Quick Start
1. `cp .env.example .env`
2. Fill Clerk and Supabase values in `.env`
3. `npm install`
4. `npm run dev`

## App URLs
- Web: `http://localhost:3000`
- API: `http://localhost:3001`
